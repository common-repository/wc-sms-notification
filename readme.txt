== Description ==

Order SMS Notofication for WooCommerce. 

## Features

- Send order updates to your customers via SMS.
- Subscribing/Opt-In SMS during checkout.
- Easy customization & tightly integrated with WooCommerce.
- Twilio sms gateway support
- Easy to use !

== Installation ==

Please note that you need to have the WooCommerce plugin installed to activate this plugin.

## Automatic Installation
1. Navigate to the WordPress Dashboard > Plugins > Add New
2. Search for 'SMS for WooCommerce'.
3. Click on Install and then Activate button.

## Manual installation
 1. Upload the entire `SMS` folder to the `/wp-content/plugins/` directory.
 2. Visit 'Plugins'.
 3. Activate the `SMS` plugin.
